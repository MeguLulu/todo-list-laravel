# todo-list-laravel

clone project :

`git clone https://github.com/VenrogMegu/todo-list-laravel.git`

Composer install :

`composer install`

Create file `.env`to configure your database, you can duplicate `.env.example`to use it as an example.

and make a `php artisan migrate`
